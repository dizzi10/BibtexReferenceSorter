For Windows 10.

To run from the command prompt:
1.) Type: cd <directory of the folder this is located in>
2.) Find your java path e.g. C:\Program Files\Java\jdk-14.0.2\bin
3.) Type: set PATH=%PATH%;C:\Program Files\Java\jdk-14.0.2\bin
4.) Type: javac.exe StableSortingWithSortingKeys.java
5.) Type: java StableSortingWithSortingKeys.java
5.) Enter input file path e.g. C:\\Users\\david\\Desktop\\bug_triage.bib
6.) Enter sorting type, "year"or "identifier"
7.) Repeat steps 4-6 if you wish to sort by another key
8.) Output should be in the same directory as this README file
